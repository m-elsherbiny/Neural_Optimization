# Neural Optimization Budgeting App

An intelligent personal budget management system powered by machine learning and mathematical optimization.

## Features
- **Smart Input**: Describe your finances in plain text - AI extracts everything.
- **Neural Classification**: MLPClassifier neural network categorizes expenses.
- **LP Optimization**: PuLP with CBC solver optimally allocates budget across goals.
- **Multi-language**: Full English and Arabic support with RTL layout.
- **Dark Mode**: Premium dark aesthetic.

## Tech Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Shadcn UI.
- **Backend**: FastAPI (Python), PuLP, Scikit-learn.
